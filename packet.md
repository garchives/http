# HTTP 报文格式

## 请求报文

```txt
┌────────────────────────────────────────────────────┐
│  Version   │ SP | Status Code | SP | Reason | CRLF | 请求行
|────────────────────────────────────────────────────|
| Field Name | :  |       Field Value         | CRLF |  请
|────────────┼────┼───────────────────────────┼──────|
| Field Name | :  |       Field Value         | CRLF |  求
|────────────┼────┼───────────────────────────┼──────|
| Field Name | :  |       Field Value         | CRLF |  头
|────────────────────────────────────────────────────|
|                         CRLF                       | 空 行
|────────────────────────────────────────────────────|
|                       Entry/Body                   | 请求体
└────────────────────────────────────────────────────┘
```

## 响应报文

```txt
┌────────────────────────────────────────────────────┐
│   Version  │ SP | Status Code | SP | Reason | CRLF | 状态行
└────────────────────────────────────────────────────┘
| Field Name | :  |       Field Value         | CRLF |  响
|────────────┼────┼───────────────────────────┼──────|
| Field Name | :  |       Field Value         | CRLF |  应
|────────────┼────┼───────────────────────────┼──────|
| Field Name | :  |       Field Value         | CRLF |  头
|────────────────────────────────────────────────────|
|                         CRLF                       | 空 行
|────────────────────────────────────────────────────|
|                       Entry/Body                   | 响应体
└────────────────────────────────────────────────────┘
```
